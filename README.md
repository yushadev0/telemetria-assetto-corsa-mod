# Telemetria Assetto Corsa Mod

Telemetria is an assistant mod designed to work alongside the Telemetria project, providing additional functionality within Assetto Corsa. <br>You can check it [here](https://github.com/yushadev0/telemetria)

## Features
- Lightweight and easy to install
- Seamless integration with Assetto Corsa Python apps system
- Designed to support Telemetria workflows

## Requirements
- Assetto Corsa (Steam version recommended)
- Windows operating system

## Installation

1. Download the latest release package.
2. Extract the `Telemetria.rar` archive.
3. Run `TelemetriaACModInstaller.exe`.
4. Select your Assetto Corsa root directory  
   (Example: `C:\Program Files (x86)\Steam\steamapps\common\assettocorsa`)
5. Click the **Install** button.
6. Launch Assetto Corsa and enable the app from the in-game Python Apps menu.

## Manual Installation (Alternative)

If you prefer manual setup:

1. Extract the archive.
2. Copy the extracted folder into: `assettocorsa/apps/python/`
3. Ensure the folder structure looks like:
```
assettocorsa/
└── apps/
    └── python/
        └── TelemetriaLB/
            ├── TelemetriaLB.py
            ├── config.ini
            ├── lb.json
            ├── map.json
            └── track_shape.json
```

## Troubleshooting

- Make sure you selected the correct Assetto Corsa directory.
- Run the installer as administrator if installation fails.
- Verify that Python Apps are enabled in Assetto Corsa settings.
